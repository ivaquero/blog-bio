зәҝжҖ§еҲҶзұ»#

1. йҖ»иҫ‘еӣһеҪ’#

1.1. жЁЎеһӢжҺЁеҜј#

иӢҘйў„жөӢеҷЁеӯҳеңЁзәҝжҖ§е…ізі»пјҢдёәдәҶдҪҝе…¶еҸҜеҜ№дәҢе…ғеҲҶзұ»иҝӣиЎҢйў„жөӢпјҢйңҖиҰҒжүҫеҲ°дёҖдёӘеҮҪж•°пјҢдҪҝеҜ№д»»ж„Ҹ\(X\)пјҢиҜҘеҮҪж•°зҡ„иҫ“еҮәйғҪеңЁ 0 е’Ң 1 д№Ӣй—ҙгҖӮйҖ»иҫ‘еӣһеҪ’пјҲlogistic regressionпјүдҪҝз”Ё sigmoid еҮҪж•°пјҡ

\[ pМӮ(X) = \frac{1}{1 + e^{-Оё^{вҠӨ}X}} \]

е…¶жәҗиҮӘеҮ зҺҮпјҲoddпјү

\[ \mathrm{odd} = \frac{pМӮ(X)}{1 - pМӮ(X)} = e^{Оё^{вҠӨ}X} \]

дёӨиҫ№еҸ–еҜ№ж•°пјҢеҸҜеҫ—еҲҶеҜ№ж•°пјҲlogitпјү

\[ \log \bigg(\frac{pМӮ(X)}{1 - pМӮ(X)} \bigg) = Оё^{вҠӨ} X = y \]

1.2. дј°и®Ўзі»ж•°#

еҜ№ејҸ\(pМӮ(X)\)жұӮеҜј

\[\begin{split} \begin{aligned} pМӮ'(y) &= \frac{e^y}{(1+e^y)^2} \\ &= pМӮ(y) * [1 - pМӮ(y)] \end{aligned} \end{split}\]

з”ұжңҖеӨ§дјјз„¶пјҢжңү

\[ P(y вҲЈ x) = pМӮ^{y} \cdot(1 - pМӮ)^{1 - y} \]

еҸ–еҜ№ж•°пјҢеҫ—

\[ J(Оё) = -\frac{1}{m} вҲ‘_{i=1}^m \bigg[y_i \log(pМӮ_i) + (1 - y_i) \log(1 - pМӮ_i)\bigg] \]

е®һйҷ…дёҠпјҢиҝҷе°ұжҳҜдәӨеҸүзҶө

иӢҘдҪҝз”ЁеҜ№ж•°жҚҹеӨұпјҢеҸӮж•°жӣҙж–°йҖҹзҺҮдјҡеҸ—еҲ° Sigmoid еҮҪж•°еҪұе“ҚпјҢжҺҘиҝ‘ 0 жҲ– 1 ж—¶дјҡеҸҳеҫ—йқһеёёж…ў

еқҸж¶ҲжҒҜжҳҜпјҢжІЎжңүе·ІзҹҘзҡ„й—ӯеҗҲеҪўејҸж–№зЁӢејҸеҸҜи®Ўз®—еҮәдҪҝиҜҘжҲҗжң¬еҮҪж•°жңҖе°Ҹзҡ„\(Оё\)еҖјгҖӮеҘҪж¶ҲжҒҜжҳҜпјҢжӯӨжҲҗжң¬еҮҪж•°жҳҜеҮёеҮҪж•°пјҢеӣ жӯӨеҸҜдҝқиҜҒжўҜеәҰдёӢйҷҚпјҲжҲ–е…¶д»–дјҳеҢ–з®—жі•пјүеҸҜжүҫеҲ°е…ЁеұҖжңҖе°ҸеҖјгҖӮеҜ№дёҠејҸжұӮеҒҸеҜј

\[ \frac{вҲӮ}{вҲӮОё_{j}}J(Оё) = \frac{1}{m} вҲ‘_{i=1}^m (Пғ(Оё^{вҠӨ}x) - y) x_{j} \]

е…¶дёӯпјҢ\(Пғ\)иЎЁзӨә Sigmoid еҮҪж•°гҖӮдёҖж—ҰжңүдәҶеҢ…еҗ«жүҖжңүеҒҸеҜјж•°зҡ„жўҜеәҰеҗ‘йҮҸпјҢе°ұеҸҜеңЁ SGD з®—жі•дёӯдҪҝз”Ёе®ғгҖӮ

2. еҲҶзұ»зҡ„еҮҶзЎ®жҖ§#

2.1. ж··ж·Ҷзҹ©йҳө#

ж··ж·Ҷзҹ©йҳөз”ЁдәҺжҰӮи§ҲеҲҶзұ»зҡ„е…·дҪ“з»“жһңгҖӮдёӢеӣҫдёәдәҢе…ғеҲҶзұ»зҡ„зӨәдҫӢпјҢе…¶еҸҜжҺЁе№ҝиҮіеӨҡзұ»еҲҶзұ»гҖӮз”ұеӣҫеҸҜзҹҘпјҢеҜ№и§’зәҝиЎЁзӨәеҫ—еҲ°жӯЈзЎ®еҲҶзұ»зҡ„е®һдҫӢдёӘж•°пјҢж•…еҜ№и§’зәҝдёҠзҡ„еҖји¶ҠеӨ§пјҢеҲҶзұ»з»“жһңи¶ҠеҘҪгҖӮж··ж·Ҷзҹ©йҳөзҡ„иЎҢгҖҒеҲ—еҲҶеҲ«иЎЁзӨәйў„жөӢеҖјгҖҒзңҹе®һеҖјгҖӮ

ж•ҙдҪ“еҲҶзұ»з»“жһңзҡ„жӯЈзЎ®зҺҮпјҲaccuracyпјүдёә

\[ \mathrm{A} = \frac{\mathrm{TP + TN}}{\mathrm{TP + TN + FP + FN}} = \frac{жӯЈеҲӨдёӘж•°}{ж ·жң¬жҖ»ж•°} \]

з”ұдәҺж ·жң¬дёҚе№іиЎЎзҡ„й—®йўҳпјҢеҜјиҮҙдәҶеҫ—еҲ°зҡ„й«ҳеҮҶзЎ®зҺҮз»“жһңеҗ«жңүеҫҲеӨ§зҡ„ж°ҙеҲҶгҖӮеҚіиӢҘж ·жң¬дёҚе№іиЎЎпјҢеҮҶзЎ®зҺҮе°ұдјҡеӨұж•ҲгҖӮ

иЎҢе’ҢдёәжҜҸдёӘзұ»еҲ«зҡ„жӯЈзЎ®жЈҖеҮәйҮҸпјҢе…¶дёӯйҳіжҖ§жӯЈзЎ®жЈҖеҮәзҺҮиў«з§°зІҫзЎ®еәҰпјҲprecisionпјүпјҢеҸҲеҸ«жөӢеҮҶзҺҮпјҡ

\[ \mathrm{P} = \frac{\mathrm{TP}}{\mathrm{TP + FP}} = \frac{зңҹйҳі}{еҲӨйҳі} \]

еҲ—е’ҢдёәжҜҸдёӘзұ»еҲ«зҡ„еҗҲи®ЎжЈҖеҮәйҮҸпјҢе…¶дёӯйҳіжҖ§еҗҲи®ЎжЈҖеҮәзҺҮиў«з§°дёәеҸ¬еӣһзҺҮпјҲrecallпјүпјҢд№ҹеҸ«зҒөж•ҸеәҰпјҲsensitivityпјүгҖҒжөӢе…ЁзҺҮгҖҒзңҹйҳіжҖ§зҺҮпјҡ

\[ \mathrm{R} = \frac{\mathrm{TP}}{\mathrm{TP + FN}} = \frac{зңҹйҳі}{е®һйҳі} \]

йҳҙжҖ§еҗҲи®ЎжЈҖеҮәзҺҮиў«з§°дёәзү№ејӮжҖ§пјҲspecificityпјүгҖҒзңҹйҳҙжҖ§зҺҮпјҡ

\[ \mathrm{S} = \frac{\mathrm{TN}}{\mathrm{TN + FP}} = \frac{зңҹйҳҙ}{е®һйҳҙ} \]

жӯӨеӨ–пјҢеҜ№дәҺзІҫзЎ®еәҰе’ҢзҒөж•ҸеәҰпјҢиҝҳжңүдёҖз§Қз®Җдҫҝзҡ„иЎЁзӨәж–№жі•пјҢ\(F_1\)еҫ—еҲҶпјҢеҚіжөӢеҮҶзҺҮе’ҢжөӢе…ЁзҺҮзҡ„и°ғе’Ңе№іеқҮж•°

\[ F_1 = \frac{2}{\mathrm{1/P + 1/R}} \]

2.2. ROC & AUC#

еҸ—иҜ•иҖ…ж“ҚдҪңзү№жҖ§пјҲreceiver-operating-characteristicпјҢROCпјүжӣІзәҝжҳҜжҳҫзӨәзҒөж•ҸеәҰпјҲзңҹйҳіжҖ§зҺҮпјҢзәөиҪҙпјүе’Ң 1-зү№ејӮжҖ§пјҲеҒҮйҳіжҖ§зҺҮпјҢжЁӘиҪҙпјүд№Ӣй—ҙе…ізі»зҡ„еӣҫиЎЁгҖӮиҖҢ ROC жӣІзәҝдёӢзҡ„з§ҜеҲҶйқўз§ҜпјҲarea under the ROCпјҢAUCпјүз”ЁдәҺжөӢйҮҸжңәеҷЁеӯҰд№ з®—жі•еҜ№дәҢиҝӣеҲ¶еҲҶзұ»й—®йўҳзҡ„жҖ§иғҪпјҲжіӣеҢ–иғҪеҠӣпјүгҖӮ

\[ \mathrm{AUC} = \frac{вҲ‘_{i=1}^{n_0} r_i - n_0 Г— (n_0 + 1)/2}{n_0 Г— n_1} \]

е…¶дёӯпјҢ\(n_0, n_1\)еҲҶеҲ«дёәеҸҚдҫӢе’ҢжӯЈдҫӢзҡ„дёӘж•°пјҢ\(r_i\)дёәз¬¬\(i\)дёӘеҸҚдҫӢеңЁж•ҙдёӘжөӢиҜ•йӣҶдёӯзҡ„жҺ’еәҸгҖӮ

зәөиҪҙ TPRпјҡи¶ҠеӨ§пјҢйў„жөӢжӯЈзұ»дёӯе®һйҷ…жӯЈзұ»и¶ҠеӨҡгҖӮ жЁӘиҪҙ FPRпјҡи¶ҠеӨ§пјҢйў„жөӢжӯЈзұ»дёӯе®һйҷ…иҙҹзұ»и¶ҠеӨҡгҖӮ

2.3. еҶізӯ–ж–№зЁӢ#

еҶізӯ–ж–№зЁӢпјҲdecision functionпјүдјҡи®Ўз®—жҜҸдёӘе®һдҫӢзҡ„еҲҶж•°пјҢиӢҘиҜҘеҲҶж•°еӨ§дәҺйҳҲеҖјпјҢеҲҷе°Ҷе®һдҫӢеҲҶй…Қз»ҷйҳіжҖ§зұ»пјӣеҸҚд№ӢпјҢе°Ҷе…¶еҲҶй…Қз»ҷйҳҙжҖ§зұ»гҖӮеҲҶж•°зҰ» 0 и¶ҠиҝңпјҢеҲҶзұ»з»“жһңзҡ„еҸҜдҝЎеәҰи¶Ҡй«ҳгҖӮйҖҡиҝҮйҷҚдҪҺеҶізӯ–ж–№зЁӢзҡ„йҳҲеҖјпјҢеҸҜеўһеҠ еҸ¬еӣһзҺҮпјҢдҪҶйңҖиҰҒжҚҹеӨұзІҫзЎ®еәҰгҖӮиҝҷеңЁеӨҡзұ»еҲҶзұ»ж—¶пјҢеҚҒеҲҶжңүз”ЁгҖӮжңәеҷЁеӯҰд№ дёӯпјҢжӢҹеҗҲеҫ—еҲ°зҡ„зі»ж•° coef_ еҚідёәеҶізӯ–ж–№зЁӢзҡ„зі»ж•°пјҢжӢҹеҗҲеҫ—еҲ°зҡ„жҲӘи·қ intercept_ еҚідёәеҶізӯ–ж–№зЁӢзҡ„еҒҸе·®гҖӮ

3. зҶө#

3.1. дҝЎжҒҜзҶө#

дҝЎжҒҜзҶөпјҲinformation entropyпјүжҳҜдҝЎжҒҜи®әзҡ„еҹәзҹіпјҢжҳҜдёҚзЎ®е®ҡжҖ§зҡ„йҮҚиҰҒеәҰйҮҸгҖӮе…¶е®ҡд№үз®Җиҝ°еҰӮдёӢпјҡ

  • зҶөпјҡеҪ“дёҖд»¶дәӢжғ…жңүеӨҡз§ҚеҸҜиғҪжғ…еҶөж—¶пјҢиҝҷд»¶дәӢжғ…еҜ№жҹҗдәәиҖҢиЁҖзҡ„дёҚзЎ®е®ҡжҖ§

  • дҝЎжҒҜпјҡиғҪеӨҹж¶ҲйҷӨиҜҘдәәеҜ№иҝҷд»¶дәӢжғ…дёҚзЎ®е®ҡжҖ§зҡ„дәӢзү©

д»Ө\(p_i\)дёәеҸҜиғҪз»“жһң\(i\)зҡ„жҰӮзҺҮпјҢеҲҷиҮӘдҝЎжҒҜдёә

\[ I_i = -\log_2 p_i \]

д№ҹе°ұжҳҜиҜҙпјҢ\((-\log_2p_i)\)дёәеҸҜиғҪз»“жһңеҸ‘з”ҹзҡ„дҝЎжҒҜйҮҸгҖӮиӢҘе…ұжңү\(c\)з§ҚеҸҜиғҪз»“жһңпјҢеҲҷзі»з»ҹзҡ„дҝЎжҒҜзҶөдёә

\[ H(X) = E(I(X)) = \frac{1}{c} \sum I_i = -\sum p_i \log_2 p_i \]

еҚізі»з»ҹзҡ„дҝЎжҒҜйҮҸзҡ„жңҹжңӣгҖӮ

1 bit дҝЎжҒҜзҶө = ln2 kJ/K зғӯеҠӣзҶөпјҲ\(k\)дёә Boltzmann еёёж•°пјү

еҜ№й«ҳж–ҜйҡҸжңәеҸҳйҮҸ\(x\)пјҢе…¶жҰӮзҺҮеҜҶеәҰеҮҪж•°дёә

\[ p(x) = \frac{1}{\sqrt{2ПҖ}Пғ} e^{\frac{-(x - Ој)^2}{2Пғ^2}} \]

еҸ–еҜ№ж•°

\[ \log p(x) = -\frac{1}{2} \log (2ПҖ) - \log Пғ - \frac{(x - Ој)^2}{2Пғ^2} \]

з”ұ\(вҲ«_{x = -вҲһ}^{вҲһ} p(x) dx = 1\)

\[\begin{split} \begin{aligned} H(X) &= -вҲ«_{x = -вҲһ}^{вҲһ} p(x)\bigg(-\frac{1}{2} \log (2ПҖ) - \log Пғ - \frac{(x - Ој)^2}{2Пғ^2}\bigg) dx \\ &= \frac{1}{2} \log (2ПҖ) вҲ«_{x = -вҲһ}^{вҲһ} p(x) dx + \log Пғ вҲ«_{x = -вҲһ}^{вҲһ} p(x) dx + \frac{1}{2Пғ^2} вҲ«_{x = -вҲһ}^{вҲһ} p(x)(x - Ој)^2 dx \\ &= \frac{1}{2} \log (2ПҖ) + \log Пғ + \frac{1}{2Пғ^2} вҲ«_{x = -вҲһ} p(x)(x - Ој)^2 dx \end{aligned} \end{split}\]

еҶҚз”ұ

\[ вҲ«_{x = -вҲһ}^{вҲһ} p(x)(x - Ој)^2 dx = рқ”ј[(x - Ој)^2] = Пғ^2 \]

еҫ—

\[\begin{split} \begin{aligned} H(X) &= \frac{1}{2} \log (2ПҖ) + \log Пғ + \frac{Пғ^2}{2Пғ^2} \\ &= \frac{1}{2} \log (2ПҖ) + \log Пғ + \frac{1}{2} \\ &= \frac{1}{2} \log (2ПҖeПғ^2) \end{aligned} \end{split}\]

еҜ№еӨҡе…ғй«ҳж–ҜеҲҶеёғ

\[\begin{split} \begin{aligned} H(X) &= \frac{1}{2} \log (2ПҖ) + \log (\det(ОЈ)) + \frac{1}{2} \\ &= \frac{1}{2} \log (2ПҖe\det(ОЈ)) \end{aligned} \end{split}\]

3.2. дәӨеҸүзҶө#

дәӨеҸүзҶөпјҲcross entropyпјүз”ЁдәҺеәҰйҮҸдёҖз»„дј°и®Ўзҡ„зұ»еҲ«жҰӮзҺҮдёҺзӣ®ж Үзұ»еҲ«зҡ„еҢ№й…ҚзЁӢеәҰгҖӮеҜ№дёӨдёӘдёҚеҗҢзҡ„еҲҶеёғ\(p, q\)еҸҜиЎЁзӨәдёә

\[ H(p, q) = -вҲ‘_i p(x) \log q(x) \]

еңЁдёҖе®ҡзЁӢеәҰдёҠпјҢзӣёеҜ№зҶөеҸҜеәҰйҮҸ 2 дёӘйҡҸжңәеҸҳйҮҸзҡ„вҖқи·қзҰ»вҖқпјҢд»Ҙе…¶еәҰйҮҸзҡ„жҲҗжң¬еҮҪж•°дёә

\[ J(Оё) = -\frac{1}{m} вҲ‘_{i=1}^m вҲ‘_{k=1}^{k} y_{ik} \log (pМӮ_{ik}) \]

е…¶дёӯпјҢ\(Оё\)дёәеӨҡдёӘ\(s_k(x)\)дёӯ\(Оё_k\)з»„жҲҗзҡ„еҸӮж•°зҹ©йҳөпјӣ\(y_{ik}\)дёәз¬¬\(i\)дёӘе®һдҫӢеұһдәҺ\(k\)зұ»зҡ„зӣ®ж ҮжҰӮзҺҮпјҢеҚі 0 жҲ– 1гҖӮеҜ№е…¶жұӮеҒҸеҜј

\[ вҲҮ_{Оё_k} J(B) = -\frac{1}{m} вҲ‘_{i=1}^m (y_{ik} - pМӮ_{ik}){x}_i \]

3.3. KL ж•ЈеәҰ#

Kullback-Leibler ж•ЈеәҰпјҲKL divergenceпјүжҳҜеәҰйҮҸдёӨдёӘеҲҶеёғжҺҘиҝ‘зЁӢеәҰзҡ„дёҖз§Қж–№жі•пјҢд№ҹз§°дёәзӣёеҜ№зҶөпјҢе…¶е®ҡд№үеҰӮдёӢ

\[\begin{split} \begin{aligned} D_\mathrm{KL}(p вҲҘ q) &:= вҲ‘p_i ((-\log_2q_i) - (-\log_2p_i))\\ &= \underbrace{-вҲ‘p_i \log q_i}_{дәӨеҸүзҶө} -\big(\underbrace{-вҲ‘p_i \log p_i}_{p зҡ„дҝЎжҒҜзҶө}\big) \\ &= вҲ‘p_i \log \frac{p_i}{q_i} \end{aligned} \end{split}\]

е…¶дёӯпјҢ\(p\)е’Ң\(q\)жҳҜдёӨдёӘжҰӮзҺҮеҲҶеёғгҖӮиҝҷиҜ»дҪңд»Һ\(q\)еҲ°\(p\)зҡ„ KL ж•ЈеәҰпјҢжҲ–\(p\)зӣёеҜ№дәҺ\(q\)зҡ„зӣёеҜ№зҶөгҖӮ

\(D_\mathrm{KL}(p вҲЈ q)\)ж•ЈеәҰеҸҜиў«и§ЈйҮҠдёәйҖҡиҝҮдҪҝз”ЁжҰӮзҺҮеҲҶеёғ\(q\)жқҘиҝ‘дјјеҲҶеёғ\(p\)иҖҢеј•е…Ҙзҡ„йўқеӨ–зҶөжҲ–дёҚзЎ®е®ҡжҖ§гҖӮ

з”ұ Gibbs дёҚзӯүејҸпјҢдәӨеҸүзҶө в©ҫ еҚ•дёӘзі»з»ҹзҶөпјҢеҪ“дё”д»…еҪ“\(p_i = q_i\)ж—¶пјҢзӯүеҸ·жҲҗз«ӢпјҢж•… KL ж•ЈеәҰ в©ҫ 0гҖӮ

дёҚйҡҫеҫ—еҲ°пјҢдәӨеҸүзҶө = дҝЎжҒҜзҶө + зӣёеҜ№зҶө

3.4. дә’дҝЎжҒҜ#

дә’дҝЎжҒҜпјҡдёӨдёӘйҡҸжңәеҸҳйҮҸ\(X, Y\)зҡ„дә’дҝЎжҒҜе®ҡд№үдёә\(X, Y\)зҡ„иҒ”еҗҲеҲҶеёғе’Ңеҗ„иҮӘзӢ¬з«ӢеҲҶеёғд№ҳз§Ҝзҡ„зӣёеҜ№зҶө

\[\begin{split} \begin{aligned} I(X;Y) &= вҲ‘_{x, y} p(x, y) \log \frac{p(x, y)}{p(x) p(y)} \\ &= H(X) + H(Y) - H(XY) \\ &= H(X) - H(X вҲЈ Y) \\ &= H(Y) - H(Y вҲЈ X) \end{aligned} \end{split}\]

ж»Ўи¶і

\[ I(X;Y) = D(P(X;Y) || P(X) P(Y)) \]

4. еӨҡе…ғйҖ»иҫ‘еӣһеҪ’#

еӨҡе…ғйҖ»иҫ‘еӣһеҪ’пјҲmultinomial logistic regressionпјүгҖӮеҪ“з»ҷе®ҡе®һдҫӢ\(x\)ж—¶пјҢйҰ–е…ҲеҲ©з”Ё sigmoid еҮҪж•°дёәжҜҸдёӘзұ»еҲ«\(k\)и®Ўз®—дёҖдёӘеҲҶж•°\(s_k(x)\)пјҢз„¶еҗҺйҖҡиҝҮеҜ№еҲҶж•°еә”з”Ё Softmax еҮҪж•°пјҲеӣ дёәе°ҶеҖјйғҪиҪ¬еҢ–дёә 0вҲј1пјүжқҘдј°и®ЎжҜҸдёӘзұ»еҲ«зҡ„жҰӮзҺҮгҖӮ

4.1. Softmax еҮҪж•°#

еҜ№\(K\)дёӘзұ»еҲ«зҡ„ Softmax еҮҪж•°еҰӮдёӢ

\[ pМӮ_k = Пғ(s(x))_k = \frac{e^{s_k(x)}}{вҲ‘_{j=1}^k e^{s_{j}(x)}} \]

е…¶дёӯ

\[ s(x) = x^{вҠӨ}Оё \]

жҳҜ\(x\)еҜ№жүҖжңүзұ»еҲ«зҡ„жҰӮзҺҮеҗ‘йҮҸпјӣ\(Пғ(s(x))_k\)жҳҜ\(x\)еҜ№зұ»еҲ«\(k\)зҡ„дј°и®ЎжҰӮзҺҮгҖӮеҚіпјҢжҜҸдёӘжҰӮзҺҮ>0пјҢжүҖжңүжҰӮзҺҮд№Ӣе’Ң=1гҖӮ

\[\begin{split} \begin{aligned} J(Оё) &= -\frac{1}{m} вҲ‘_{i=1}^m\bigg(\log \frac{e^{s_k(x)}}{вҲ‘_{j=1}^k e^{s_{j}(x)}}\bigg)\\ &= -\frac{1}{m} вҲ‘_{i=1}^m \bigg(\log e^{s_k(x)} + \log вҲ‘_{j=1}^k e^{s_{j}(x)}\bigg)\\ &= -\frac{1}{m} вҲ‘_{i=1}^m \bigg(s_k(x) + \log вҲ‘_{j=1}^k e^{s_{j}(x)} \bigg) \end{aligned} \end{split}\]

е…¶дёӯпјҢз¬¬дёҖйЎ№жҳҜз¬¬\(k\)дёӘзұ»еҲ«иў«йў„жөӢжӯЈзЎ®зҡ„жҰӮзҺҮпјҢз¬¬дәҢйЎ№дёә\(K\)дёӘзұ»еҲ«зҡ„жҖ»жҰӮзҺҮгҖӮ

Softmax еҲҶзұ»еҷЁдёҖж¬ЎеҸӘиғҪйў„жөӢдёҖдёӘзұ»еҲ«пјҲе®ғжҳҜеӨҡзұ»еҲ«пјҢдҪҶдёҚжҳҜеӨҡиҫ“еҮәпјүпјҢеӣ жӯӨе®ғеҸӘиғҪдёҺдә’ж–Ҙзұ»еҲ«дёҖиө·дҪҝз”ЁпјҢиҖҢж— жі•з”ЁдәҺеңЁдёҖеј з…§зүҮдёӯиҜҶеҲ«еӨҡдёӘдәәгҖӮ

4.2. жңҖеӨ§зҶөеҺҹзҗҶ#

зӣҙи§Ӯең°иҜҙпјҢдёҖдёӘеҲҶеёғи¶ҠеҲҶж•ЈпјҢе…¶зҶөе°ұи¶ҠеӨ§гҖӮеҜ№дәҺе®Ңе…ЁйҡҸжңәзҡ„еҸҳйҮҸпјҲзӯүеҸҜиғҪпјүпјҢдҝЎжҒҜзҶөжңҖеӨ§гҖӮеҒҮи®ҫж•°жҚ®жҳҜзҰ»ж•ЈеҲҶеёғзҡ„пјҢ\(k\)дёӘзү№еҫҒзҡ„жҰӮзҺҮеҲҶеҲ«дёә\(p_k\)пјҢжңҖеӨ§зҶөеҺҹзҗҶпјҲmaximum entropy principleпјүеҸҜд»ҘиЎЁиҝ°дёәпјҡ

\[\begin{split} \max\{H(p)\} = \min\bigg\{вҲ‘_{k=1}^k p_k\log p_k\bigg\}\\ s.t.\ вҲ‘_{k=1}^k p_k = 1 \end{split}\]

еҲ©з”Ё Lagrange д№ҳеӯҗжі•пјҡ

\[ L(p,О») = вҲ‘_{k=1}^k p_k\log p_k + О»\bigg(1 - вҲ‘_{k=1}^k p_k\bigg) \]

дәҺжҳҜеҸҜеҫ—пјҡ

\[ p_1 = p_2 = вӢҜ = p_k = \frac{1}{k} \]

еӣ жӯӨзӯүеҸҜиғҪзҡ„жғ…еҶөзҶөжңҖеӨ§гҖӮ

4.3. жқЎд»¶зҶө#

\[\begin{split} \begin{aligned} H(Y вҲЈ X) &:= -вҲ‘_{x, y} p(x) p(y вҲЈ x) \log p(y вҲЈ x) \\ &= E(H(Y вҲЈ X = x^{(k)})) \end{aligned} \end{split}\]

жұӮжқЎд»¶зҶөжңҖеӨ§пјҢеҚіжұӮ

\[\begin{split} \min_{x, y} вҲ‘_{x, y} \tilde{p}(x) p(y вҲЈ x) \log p(y вҲЈ x)\\ s.t. \begin{cases} &О”_k &- E_p(f_k(x, y)) &= 0, &k = 1, 2 вҖҰ, m \\ &1 &- вҲ‘_{y} p(y вҲЈ x) &= 0 \end{cases} \end{split}\]

еҲ©з”Ё Lagrange д№ҳеӯҗжі•пјҡ

\[\begin{split} \begin{aligned} L(P, О») &= вҲ‘_{x, y} \tilde{P}(x) P(y вҲЈ x) \log P(y вҲЈ x)\\ & + О»_0\bigg(1- вҲ‘_y P(y вҲЈ x)\bigg) + вҲ‘_{k=1}^m О»_k(О”_k-E_P(f_k(x, y))) \end{aligned} \end{split}\]

еҸҜеҫ—

\[ P(y вҲЈ x)=\frac{e^{О·^{вҠӨ}вӢ…f(x, y)}}{вҲ‘_{y} e^{О·^{вҠӨ}вӢ…f(x, y)}} \]

е…¶дёӯпјҢ\(О·\)дёәеёёж•°еҗ‘йҮҸгҖӮ

з”ұдёҠпјҢsigmoid е’Ң Softmax з¬ҰеҗҲжңҖеӨ§зҶөеҺҹзҗҶгҖӮ

5. еӨҡзұ»еҲҶзұ»#

дёҖдәӣз®—жі•пјҢеҰӮйҡҸжңәжўҜеәҰдёӢйҷҚпјҲSGDпјүпјҢйҡҸжңәжЈ®жһ—пјҲRFпјүе’Ңжңҙзҙ иҙқеҸ¶ж–Ҝжң¬иә«пјҲnBпјүе°ұеҸҜеӨ„зҗҶеӨҡдёӘзұ»еҲ«зҡ„еҲҶзұ»пјҢеұһдәҺеӨҡзұ»еҲҶзұ»еҷЁпјҲmulti-class classifiersпјүгҖӮиҖҢйҖ»иҫ‘еӣһеҪ’жҲ– SVM дёҘж јжқҘиҜҙжҳҜдәҢе…ғеҲҶзұ»еҷЁпјҲbinary classifiersпјүгҖӮиҰҒдҪҝз”ЁдәҢе…ғеҲҶзұ»еҷЁеӨ„зҗҶеӨҡзұ»еҲҶзұ»пјҢйңҖиҰҒйҮҮеҸ–дёҖдәӣжҲҳз•ҘгҖӮ

дёҖеҜ№дёҖпјҲone-versus-oneпјҢOvOпјүд№ҹз§°е…Ёй…ҚеҜ№пјҲall-pairsпјүпјҢе…¶е°Ҷ\(n\)дёӘзұ»еҲ«дёӨдёӨй…ҚеҜ№пјҢдә§з”ҹ\(\binom{n}{2}\)дёӘдәҢе…ғеҲҶзұ»еҷЁгҖӮйҡҸеҗҺиҝӣиЎҢеҰӮдёӢжӯҘйӘӨпјҡ

  • ж–°ж ·жң¬жҸҗдәӨз»ҷжүҖжңүеҲҶзұ»еҷЁйў„жөӢпјҡ\(\binom{n}{2}\)дёӘеҲҶзұ»з»“жһң

  • жҠ•зҘЁдә§з”ҹжңҖз»ҲеҲҶзұ»з»“жһңпјҡиў«йў„жөӢжңҖеӨҡзҡ„зұ»еҲ«дёәжңҖз»Ҳзұ»еҲ«

  • ж–°ж ·жң¬жҸҗдәӨз»ҷжүҖжңүеҲҶзұ»еҷЁйў„жөӢпјҡ\(n\)дёӘеҲҶзұ»з»“жһң

  • жҜ”иҫғеҗ„еҲҶзұ»еҷЁзҡ„йў„жөӢзҪ®дҝЎеәҰпјҡ

еӨҡеҜ№еӨҡеҜ№еҲҶзұ»еҷЁй”ҷиҜҜжңүдёҖе®ҡе®№еҝҚе’Ңдҝ®жӯЈиғҪеҠӣпјҢзј–з Ғи¶Ҡй•ҝгҖҒзә й”ҷиғҪеҠӣи¶ҠејәпјӣеҜ№еҗҢзӯүй•ҝеәҰзҡ„зј–з ҒпјҢзҗҶи®әдёҠжқҘиҜҙпјҢд»»ж„ҸдёӨдёӘзұ»еҲ«д№Ӣй—ҙзҡ„зј–з Ғи·қзҰ»и¶ҠиҝңпјҢеҲҷзә й”ҷиғҪеҠӣи¶ҠејәгҖӮ

6. еӨҡиҫ“еҮәеҲҶзұ»#

иҖғиҷ‘дёҖдёӘдәәи„ёиҜҶеҲ«еҲҶзұ»еҷЁпјҡиӢҘиҜҶеҲ«еҗҢдёҖеј еӣҫзүҮдёӯзҡ„еӨҡдёӘдәәиҜҘжҖҺд№ҲеҠһпјҹжҜҸдёӘиҜҶеҲ«зҡ„дәәйғҪеә”иҜҘйҷ„дёҠдёҖдёӘж ҮзӯҫгҖӮиҝҷз§Қиҫ“еҮәеӨҡдёӘдәҢе…ғж Үзӯҫзҡ„еҲҶзұ»зі»з»ҹз§°дёәеӨҡж ҮзӯҫеҲҶзұ»пјҲmulti-label classificationпјүзі»з»ҹгҖӮ

иҜ„дј°еӨҡж ҮзӯҫеҲҶзұ»еҷЁзҡ„ж–№жі•жңүеҫҲеӨҡпјҢйҖүжӢ©жӯЈзЎ®зҡ„жҢҮж ҮзЎ®е®һеҸ–еҶідәҺжӮЁзҡ„йЎ№зӣ®гҖӮдёҖз§Қж–№жі•жҳҜжөӢйҮҸжҜҸдёӘеҚ•зӢ¬ж Үзӯҫзҡ„ F1 еҫ—еҲҶпјҢз„¶еҗҺз®ҖеҚ•ең°и®Ўз®—е№іеқҮеҲҶж•°гҖӮиҝҷдёӘеҒҮи®ҫжҳҜжүҖжңүж ҮзӯҫйғҪеҗҢж ·йҮҚиҰҒпјҢдҪҶпјҢдәӢе®һе№¶йқһеҰӮжӯӨгҖӮдёҖз§Қз®ҖеҚ•зҡ„йҖүжӢ©жҳҜз»ҷжҜҸдёӘж ҮзӯҫиөӢдәҲзӯүдәҺе…¶ж”ҜжҢҒзҡ„жқғйҮҚпјҲеҚіе…·жңүиҜҘзӣ®ж Үж Үзӯҫзҡ„е®һдҫӢж•°пјүгҖӮ

еӨҡиҫ“еҮәеҲҶзұ»пјҲmulti-output classificationпјүеҸӘжҳҜеӨҡж ҮзӯҫеҲҶзұ»зҡ„жҺЁе№ҝпјҢе…¶дёӯжҜҸдёӘж ҮзӯҫеҸҜжҳҜеӨҡзұ»пјҲеҚіпјҢе®ғеҸҜе…·жңүдёӨдёӘд»ҘдёҠзҡ„еҸҜиғҪеҖјпјүгҖӮеӨҡиҫ“еҮәзі»з»ҹдёҚйҷҗдәҺеҲҶзұ»д»»еҠЎпјӣе®ғиҝҳеҸҜз”ЁдәҺеӨҡдёӘд»»еҠЎгҖӮжӮЁз”ҡиҮіеҸҜиғҪжӢҘжңүдёҖдёӘзі»з»ҹпјҢиҜҘзі»з»ҹжҜҸдёӘе®һдҫӢиҫ“еҮәеӨҡдёӘж ҮзӯҫпјҢеҢ…жӢ¬зұ»ж Үзӯҫе’ҢеҖјж ҮзӯҫгҖӮ
